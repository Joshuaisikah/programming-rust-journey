# Chapter 8 Project: Multi-Module CLI Tool Suite

Build a CLI tool with multiple commands in separate modules.

```bash
cargo run encode "hello"     # Base64 encoder
cargo run hash "file.txt"    # File hasher
cargo run compress "data"    # Compression
```

## Features

- Separate module per command
- Shared utilities module
- Public API design
- Re-exports

Master Rust project structure! 📦
