# Chapter 1: Systems Programmers Can Have Nice Things

## Overview
This chapter introduces Rust and explains why it was created. You'll learn about the problems Rust solves and get a preview of its key features.

## Learning Objectives
- ✅ Understand memory safety issues in C/C++
- ✅ Learn how Rust prevents undefined behavior
- ✅ Get introduced to the ownership system
- ✅ Recognize Rust's performance guarantees

## What to Build
1. **Hello World** ✅ (already done)
2. **Memory safety examples** - Show why Rust is safe
3. **Performance comparison** - Benchmark Rust vs other languages

## Key Takeaways from Chapter
- Rust provides memory safety WITHOUT garbage collection
- Zero-cost abstractions - you don't pay for what you don't use
- Ownership system prevents entire classes of bugs at compile time
- Rust is as fast as C/C++ but much safer
